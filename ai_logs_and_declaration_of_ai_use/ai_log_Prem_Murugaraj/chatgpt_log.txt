1. Data Loading (loader.py)

ISD-Lite files are whitespace-separated.

A loading function could look like:

import pandas as pd

COLUMN_NAMES = [
    "year",
    "month",
    "day",
    "hour",
    "air_temperature",
    "dew_point",
    "sea_level_pressure",
    "wind_direction",
    "wind_speed",
    "sky_condition",
    "precipitation_1h",
    "precipitation_6h",
]

def load_weather_data(filepath: str) -> pd.DataFrame:
    """
    Load NOAA ISD-Lite weather data.

    Parameters
    ----------
    filepath : str
        Path to ISD-Lite file.

    Returns
    -------
    pandas.DataFrame
    """

    df = pd.read_csv(
        filepath,
        delim_whitespace=True,
        names=COLUMN_NAMES,
    )

    return df
2. Data Cleaning (cleaning.py)

NOAA uses special missing values.

For temperature:

-9999 = missing

Temperature is stored in tenths of degrees Celsius.

import pandas as pd

MISSING_TEMP = -9999

def clean_temperature(df: pd.DataFrame) -> pd.DataFrame:
    """
    Replace missing values and convert temperature to Celsius.
    """

    df = df.copy()

    df["air_temperature"] = (
        df["air_temperature"]
        .replace(MISSING_TEMP, pd.NA)
        / 10
    )

    return df

You can also remove rows:

df = df.dropna(subset=["air_temperature"])

def create_datetime(df):

    return df

Yes. The downloader should be its own module (e.g. loader.py). Given a station ID and a year, it should:

Build the correct NOAA URL.
Download the .gz file.
Save it locally (or optionally keep it in memory).
Return the path to the downloaded file.

The NOAA ISD-Lite data is organized by year, and the filename format is:

https://www.ncei.noaa.gov/pub/data/noaa/isd-lite/<YEAR>/<STATION>-<YEAR>.gz

For example:

https://www.ncei.noaa.gov/pub/data/noaa/isd-lite/2023/010010-99999-2023.gz

This directory structure and filename pattern are used throughout the ISD-Lite archive.

Step 1: Install requests

If you're using uv:

uv add requests
Step 2: loader.py
from pathlib import Path
import requests

BASE_URL = "https://www.ncei.noaa.gov/pub/data/noaa/isd-lite"


def download_weather_data(
    station: str,
    year: int,
    output_dir: str = "data",
) -> Path:
    """
    Download a NOAA ISD-Lite file.

    Parameters
    ----------
    station : str
        Weather station ID (e.g. 010010-99999).
    year : int
        Year of the data.
    output_dir : str
        Directory where the file will be saved.

    Returns
    -------
    Path
        Path to the downloaded .gz file.

    Raises
    ------
    FileNotFoundError
        If the station or year does not exist.
    """

    output_path = Path(output_dir)
    output_path.mkdir(parents=True, exist_ok=True)

    filename = f"{station}-{year}.gz"
    url = f"{BASE_URL}/{year}/{filename}"

    destination = output_path / filename

    response = requests.get(url, timeout=30)

    if response.status_code == 404:
        raise FileNotFoundError(
            f"No data found for station {station} in {year}."
        )

    response.raise_for_status()

    destination.write_bytes(response.content)

    return destination

Uses pathlib (recommended for modern Python).
Raises meaningful exceptions instead of printing errors.
Automatically creates the data/ directory if it doesn't exist.
Returns the downloaded file path so the next function can read it.

The cleaning step should only prepare the data for analysis. For NOAA ISD-Lite temperature data, this typically means:

Replace missing values (-9999) with NaN.
Convert temperature from tenths of °C to °C.
Remove rows where temperature is missing.
Optionally sort by datetime.
cleaning.py
import pandas as pd

# NOAA missing value for air temperature
MISSING_TEMPERATURE = -9999


def clean_temperature(df: pd.DataFrame) -> pd.DataFrame:
    """
    Clean the temperature data.

    Parameters
    ----------
    df : pandas.DataFrame
        DataFrame containing 'datetime' and 'temperature'.

    Returns
    -------
    pandas.DataFrame
        Cleaned DataFrame with temperatures in degrees Celsius.
    """

    cleaned = df.copy()

    # Replace NOAA missing values with NaN
    cleaned["temperature"] = cleaned["temperature"].replace(
        MISSING_TEMPERATURE,
        pd.NA,
    )

    # Convert from tenths of degrees Celsius to degrees Celsius
    cleaned["temperature"] = cleaned["temperature"] / 10

    # Remove rows with missing temperatures
    cleaned = cleaned.dropna(subset=["temperature"])

    # Ensure chronological order
    cleaned = cleaned.sort_values("datetime").reset_index(drop=True)

    return cleaned

For a weather data assignment, I'd keep the analysis focused on statistics that are meaningful but not overly complex. You want enough to demonstrate data processing without making the project too large.

Basic Statistics (Definitely Include)

Maximum temperature
Mean temperature
Median temperature

def analyze_temperature(df):

Additional Statistics (Good Additions)
Temperature Range

Shows the spread between the warmest and coldest observations.

range = max - min
Quartiles

Useful for understanding the distribution.

Q1 (25%)
Median (50%)
Q3 (75%)
temp.quantile([0.25, 0.5, 0.75])
Variance

Since you're already computing the standard deviation, you can also report the variance.

variance = temp.var()
Missing Values

Since you're cleaning the data, it's useful to report how many values were removed.

For example:

Original observations : 8760
Removed observations  : 15
Remaining observations: 8745

This demonstrates that your cleaning step had an effect.

Time-Based Analysis

Since you have a datetime column, you can compute more interesting summaries.

Monthly Average Temperature
monthly = (

This is particularly useful if you're plotting the data.

Daily Average Temperature
daily = (
    df.groupby(df["datetime"].dt.date)["temperature"]
      .mean()

This creates one average temperature per day and can make a year's worth of hourly data much easier to visualize.
